Section 01

One-Page Product Vision

The Promise

Quickpost is the social media manager that already knows your business. Show up once, post forever — with studio-quality images and brand-specific copy, generated in seconds.

Core Insight: The gap between generic AI content and premium AI content isn't model quality — it's context depth. Pomelli feels smart because it refuses to generate until it knows enough. Quickpost must do the same. And now with AI Photo Studio, it transforms any basic product photo into a marketing-ready image that matches the brand.

Law 01

Never generate into a vacuum

Always have enough brand signal before producing. Confidence gates enforce this automatically.

Law 02

Specificity over polish

"Summer haircuts at Mane Street in Cedar Rapids" beats "Transform your look this season" every time.

Law 03

Done-for-you, not fill-in-the-blank

The user approves and publishes. They don't assemble, configure, or prompt.

Law 04

Studio quality, zero effort

Upload a raw product photo. Receive a marketing-ready image. No design skills required.

Section 05

AI Generation Pipeline

Five stages. All hidden behind the loading experience. Each stage has its own quality gate before proceeding.

S1
Brief Assembly

Merge Brand Brain + user intent + seasonal hook + local signal into a structured creative brief (JSON). Minimum confidence threshold check runs here.

Gate: conf ≥ 0.6
S2
Concept Generation

Generate 3 distinct post angles: [angle_name, hook, rationale, format_recommendation]. Anti-generic similarity filter runs here.

Gate: similarity filter
S3
Caption Generation

Write caption with local specificity injected. Short + long variants. Hashtag set. Voice fingerprint checked against brand_vibe profile.

Gate: voice match ≥ 70%
S4
Image Concept / Prompt Generation

Generate detailed, art-directed image prompt. Style direction + color guidance from Visual Layer. Anti stock-photo-ism check.

Gate: specificity check
S5
Quality Score + Retry Controller

Score output 0–100 across 5 dimensions. If ≥ 80: deliver. If 60–79: silent retry on lowest dimension. If < 60: full retry with tighter constraints.

Deliver if ≥ 80/100

A small business owner photographs their product on a kitchen table. Within 30 seconds, Quickpost returns a set of marketing-ready images with professional backgrounds, lighting, and brand-consistent styling — no photographer, no studio, no design software.

What Happens in the Background

Photo Studio AI Pipeline · hidden from user
1
Image Analysis Identify subject type (product, food, service space, team member), dominant colors, current background quality, key object to preserve.
Vision API → subject classification JSON
2
Subject Isolation Background removal using segmentation model. Preserve subject edges, shadows, reflections where they add realism.
Segmentation model → masked subject PNG
3
Brand Style Context Assembly Pull from Brand Brain: brand_colors, visual_style, studio_style_preference. Build style-direction prompt specific to this business.
Brand Brain → styled image-to-image prompt
4
Scene Generation (Image-to-Image) Generate new background/environment matched to selected style. Place isolated subject. Adjust lighting consistency. Apply brand color harmony.
Image-to-image model → 4 variants generated
5
Post-Processing + Format Export Auto-crop for IG (1:1, 4:5) and FB (16:9, 1:1). Quality check: resolution, edge artifacts, brand color accuracy. Return 4 selectable variants.
Image quality gate → multi-format export
6
Caption Auto-Generation (paired) Generate a caption specific to the transformed image. Caption references the product/subject just shown — not a generic post. Pair and deliver together.
Caption pipeline with image-aware prompt

Photo Studio Quality Controls

CheckRuleAction if Fail
Edge artifact detection No visible halo or jagged edges around subject Re-run segmentation with higher precision; retry generation
Brand color harmony Output image incorporates ≥1 brand color from Brand Brain Adjust color grading in post-processing pass
Lighting consistency Shadow direction and light source consistent across subject + background Flag to user: "Lighting may look off — regenerate?"
Subject identity preservation Core subject unchanged — no invented details, distortions, color changes Hard block: reject generation, retry with subject-lock instructions
Resolution gate Output ≥ 1080px on shortest side Upscale pass before delivery
No hallucinated objects Background contains no invented logos, text, or identifiable real locations Retry with explicit negative prompt: "no text, no logos, no landmarks"

Three Photo Input Types Supported

🏠

Interior shots. Enhancement mode: improve lighting, color grading, atmosphere. No subject isolation needed.

Key System Prompts

Brief Assembly Prompt (System)

"You are a strategic social media director for local businesses. You receive a Brand Brain JSON and user post intent. Output a creative brief hyper-specific to this exact business in this exact city. Never generate anything that could apply to a competitor. Always anchor to the business's name, location, and actual services."

Photo Studio Analysis Prompt

"Analyze this image. Return JSON: {subject_type, primary_subject_description, background_quality, dominant_colors, preservation_notes, recommended_styles[]}. Never invent details not visible in the image. The preservation_notes field must describe exactly what must not change in the output."

Anti-Generic Caption Rule (enforced in prompt)

"Before finalizing any caption: verify it contains the business name OR a specific service name, a local geographic reference, and no phrases from the banned openers list. If any check fails, rewrite completely."
